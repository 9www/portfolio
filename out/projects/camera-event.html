Camera Event
Take a photo and save it
Type : Interactive Camera
Year : 2021
Programming language : C#
Engine: Unity
Description :
This project is kontec interactive project. Build 're-use' conponents in UnityC# engine, that can reduce develop time. Interactive camera X-Mas event, users can take a photo with cartoon character. After a second, the photo will upload to server, also the server will return the link and generate the qr code at the display monitor. Lastly Users can screen qr code and download the photo.
